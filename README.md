Food Order Management System
Overview

The Food Order Management System is a web-based application developed using ASP.NET Web Forms and Oracle Database. The system was designed to digitize and manage internal food ordering operations within an organization. It replaces manual processes with a centralized platform for order placement, tracking, and administrative control.

Features

Role-based login authentication

Employee dashboard for placing and tracking orders

Admin dashboard for monitoring and managing orders

Food item management

Centralized order records management

Session handling and access control

System Architecture

The application follows a three-tier architecture:

Presentation Layer – ASP.NET Web Forms

Business Logic Layer – C#

Data Layer – Oracle Database

This structure ensures separation of concerns, maintainability, and secure data handling.

Technologies Used

ASP.NET Web Forms

C#

Oracle Database

HTML, CSS

Visual Studio

Git and GitHub

Project Objective

The objective of this project was to develop a structured and secure web application that improves operational efficiency, reduces manual errors, and provides reliable record management for internal food ordering processes.
